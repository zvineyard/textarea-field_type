### Configuration

Below is the full configuration available with defaults values:

    "example" => [
        "type"   => "anomaly.field_type.textarea",
        "config" => [
            "rows"          => 6,
            "min"           => null,
            "max"           => null,
            "default_value" => null,
            "show_counter"  => true,
            "autogrow"      => true,
        ]
    ]
